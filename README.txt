┃
┣━社会に問いかけたいこと
┃　 │
┃　 └私は男性差別が嫌いだ。
┃　  私は高校は男子校に通っていたのだが、下校中、同級生の男子が、電
┃　  他校の女子複数に盗撮されたるという事件が起こった。制服から学校
┃　  が特定できたので、その学校に苦情入れる準備を整えているとき、被
┃　  害者に諦めた様子で止められた。性犯罪では男が女から被害を受けた
┃　  場合、何もできないのだということを深く思い知ることとなった。
┃　  それから毎日駅のホームで「女性専用車」を見るたびに、怒りと屈辱
┃　  を感じるようになった。女性だけが性犯罪の被害者であるわけではな
┃　  い。男性の被害はカウントされてないだけで、実際は性犯罪率は、男
┃　  女でそれほど変わらないのかもしれない。女性専用車は、その上で男
┃　  性であるというだけで私たちを性犯罪者予備軍として締め出す。
┃　  ほとんど同学年で同い年のはずのあの盗撮女たちが立ち入ることがで
┃　  きて、私たちが立ち入れない空間。何が違うかってただ性別が違うだ
┃　  け。それが女性専用車なのである。
┃　  女性専用車を見ないようにしても、駅のアナウンスから女性専用車の
┃　  案内放送が聞こえてくる。それじゃなくても、私たちは電車内では常
┃　  に痴漢のでっちあげに怯えなくてはならない。「差別」と「屈辱」を
┃　  毎日、同時に味わわされるのである。
┃　  差別や屈辱から逃げるのはもうやめよう。私が高校で男性差別反対を
┃　  唱えたとき、「昔は男尊女卑だったから今はある程度しょうがない」
┃　  とか言った男子がいる。これは逃げである。「今」しか生きていない
┃　  私たちは一度も男尊女卑や女性差別を許された覚えはない。少なくと
┃　　も目に見える形では。私たちは差別されっぱなしであることに目を背
┃　　けてはいけない。

研究・開発したいこと…活用可能性が高く、
┃　　　　　　　　　　社会に何かを訴えかけるきっかけと成り得る技術
┣━3Dプリント
┃　││
┃　│└巨大だが「0」と「1」しか通れない道路に過ぎないはずのITは、様々
┃　│ な人が様々な想いやデータをその｢0｣と｢1｣に託すことで、
┃　│ IComunicationTやIObjectTを生み、軍事、医療、金融、政治、
┃　│ イベント・・・様々なものを大きく変えた。
┃　│ この道路は不思議だ。｢0｣と｢1｣になるまで分解することさえできれば、
┃　│ 動画でも、紙に印刷された資料でも、一瞬でどこにでも送ることがで
┃　│ きる。こうして思想、教育、ヒト、モノ、カネ、情報を世界中に広め
┃　│ ることができる。
┃　│ 動画、印刷物がこの道路を通れるなら、立体だって通れるだろう。こ
┃　│ の立体は、思想を形にすることに使えるかもしれない。教育を円滑に
┃　│ 進める触媒となるかもしれない。この立体というモノを求めてヒトが
┃　│ 集まり、カネが動くかもしれない。3Dスキャンして、それを遠くで3D
┃　│ プリントすれば、「どこでもドア」が作れるかもしれない。何ができ
┃　│ るかわからない。だからこそワクワクしているのだ。
┃　│
